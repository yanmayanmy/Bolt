# Bolt
Slack API Tutorial.

## Launch
### Starting App
```
$ node app/app.js
```
### Publish URL
```
$ ngrok http 8000
```

Go to "Event Subscription" from your Slack App dashboard.

Copy & Paste the URL that's been created by ngrok to the Request URL column.

## Scope
### Bot events
- `message.channels`

Reinstall the app if needed and bot will start working.
